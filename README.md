🌍 Wanderlust
📖 Overview
Wanderlust is a full-stack web application inspired by the popular vacation rental platform, Airbnb. This project allows users to explore listings, create accounts, and manage their rental properties. With a modern interface and robust backend, Wanderlust integrates essential features like authorization, file storage, map displays, and responsive design. The project follows the MVC architecture and employs RESTful APIs for seamless communication between the frontend and backend.

🌟 Features
User Authentication:
    Secure login and signup functionality.
    Use of web cookies and session-based authentication.
Listings Management:
    Authorization for creating, editing, and deleting property listings.
    Integration with cloud storage for uploading and managing property images.
Map Display:
    Powered by Mapbox for interactive maps and precise location visualization.
Flash Messages:
    Friendly and informative flash messages for user feedback.
Responsive UI:
    Built with Bootstrap for mobile-first, responsive design.
Database Management:
    Hosted on MongoDB Atlas for scalable and secure data storage.

🚀 Technologies Used
Frontend: 
    HTML5, CSS3, and JavaScript
    Bootstrap for responsive design and styling.
Backend:
    Node.js and Express.js for server-side functionality.
    Mongoose for MongoDB object modeling.
Other Tools & Services:
MongoDB Atlas: Cloud database service.
Render: Deployment platform for the project.
Mapbox: API for map integration.
Multer & Cloud Storage: For handling file uploads to the cloud.
Express Middlewares: For handling authentication, validation, and error management.

🎯 Architecture
Wanderlust follows the MVC (Model-View-Controller) pattern for maintainable and scalable code:

Model: Defines the database schemas using Mongoose.
View: Frontend templates styled with Bootstrap.
Controller: Handles the application logic and interaction with the database.
